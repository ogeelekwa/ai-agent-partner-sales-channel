# Changelog

All notable changes to this project will be documented in this file.

---

## [1.0.0] - 2026-02-09

### Added

- Initial release of the Shopify Sales Channel template
- React Router 7.x integration with Shopify App Bridge
- Drizzle ORM with SQLite for session storage
- Product publishing functionality via `ProductPublishingCard` component
- Internationalization support (i18next) with English, Spanish, Thai, and Turkish translations
- Webhook handlers for `app/uninstalled` and `app/scopes_update`
- MCP (Model Context Protocol) configuration for Cursor, GitHub Copilot, Claude Code, and Gemini CLI
- Polaris UI components integration
- GraphQL codegen setup for Shopify Admin API (2026-04)
- Docker support with included `Dockerfile`
